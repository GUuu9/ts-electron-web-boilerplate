# 📅 Changelog: 2026-05-13 (수)

## 🌟 주요 업데이트

### 🧠 Aiming AI 시스템 고도화 (Behavior Tree 기반)
타겟 추적 및 물리 예측 기반의 지능형 조준 시스템을 구현했습니다.

- **물리 시뮬레이션**: 중력 가속도를 고려한 포물선 궤적 계산 및 벽/천장 반사(Bounce) 물리 엔진 구현.
- **예측 조준**: 타겟의 이동 벡터(vx, vy)와 예상 비행 시간을 기반으로 하는 Linear Prediction 조준 알고리즘 적용.
- **자기 수정형 학습(Self-Correcting Learning)**:
  - **좌표 그리드 기반 저장소**: 타겟의 위치(40x40 그리드)별 최적 조준값(Angle, Power)을 저장.
  - **성공 기반 학습**: 명중 시 해당 파라미터를 저장하고 가중치(Weight)를 높여 신뢰도를 향상.
  - **강제 재학습(Forced Exploration)**: 명중 실패 시 해당 위치의 잘못된 데이터를 즉시 삭제하고, 타겟 방향 기반의 새로운 파라미터를 생성하여 재조준 유도.
  - **적응형 학습률**: 신뢰도가 낮은 초기 단계에서는 과감한 파라미터 보정을, 높은 단계에서는 세밀한 수정을 수행.

- **사용자 제어 및 시각화**:
  - **UI 제어 패널**: 타겟의 이동(Move)과 명중 후 자동 재배치(Relocate)를 UI 상에서 버튼으로 토글 가능.
  - **상태 시각화**: 가중치에 따른 투사체 색상 보간(파란색 → 녹색)으로 학습 정도를 실시간 표시.
  - **궤적 및 상태 가이드**: 발사 시점에 예상 포물선 궤적을 렌더링하고, 실시간 조준 파라미터(Angle, Power, Learned Count) 표시.
  - **성능 최적화**: 매 프레임 렌더링하던 궤적을 발사 시점에만 수행하도록 수정하고 배속(1x~5x) 연산 통합.
