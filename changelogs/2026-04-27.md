# 📅 2026-04-27 (월) 변경 이력

## 🚀 주요 업데이트

### 1. 전사적 보안 인프라 구축 (Shared Security Module)
플랫폼에 독립적으로 작동하는 고수준 보안 및 데이터 처리 모듈을 `src/shared/security`에 구축했습니다.
- **AES-256-GCM**: 대칭키 암호화 구현 (Web Crypto API 기반).
- **RSA-OAEP**: 비대칭키 암호화 및 키 교환 로직 구현 (1024/2048/4096 bits 지원).
- **Multi-Algorithm Compression**: 전송 효율화를 위한 Gzip, Deflate, Deflate-Raw 압축 지원.

### 2. 보안 테스트 페이지 (Renderer UI) 추가
새롭게 구현된 보안 기능들을 실시간으로 검증할 수 있는 UI 페이지를 추가했습니다.
- **AES Tester**: 32바이트 입력 제한 및 랜덤 키 생성기(Random Generator) 포함.
- **RSA Tester**: 키쌍 생성 및 전체 키 데이터(Full Text) 노출 기능 포함.
- **Compression Tester**: 알고리즘별 압축률 실시간 계산 및 시각화.

### 3. 가이드 문서 및 구조 최적화
- **SECURITY_GUIDE.md**: 보안 인프라 사용 가이드 작성.
- **Layered Architecture**: Shared 도메인 내부의 체계적인 관리를 위해 `security` 폴더 격리 및 DI 주입 완료.
