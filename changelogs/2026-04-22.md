## 🛠 상세 변경 내역

### 1. OS Core 인프라 구축
- `OSIntegrationService`를 메인 DI 컨테이너에 등록하여 전역에서 관리.
- `src/main.ts`에서 앱 창 생성 시 OS 기능을 자동 초기화하도록 로직 추가.
- `app.setAppUserModelId` 설정을 통해 Windows 10/11 알림 센터와의 호환성 확보.

### 2. IPC 브릿지 및 UI 연결
- `src/preload.ts`에 `electronAPI.os.notify` 메서드 노출.
- `Maintenance > OS Integration` 탭 UI 신설 및 `MaintenanceController`에 알림 전송 로직 구현.

### 3. 안정성 및 디버깅 강화
- 알림 전송 시 메인 프로세스에서 상세 로그(Attempt, Shown)를 출력하여 문제 추적 용이성 확보.
- TypeScript 타입 오류(`error` 이벤트 미지원 등)를 Electron 표준 사양에 맞춰 수정.
