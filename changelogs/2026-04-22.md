# 📝 개발 로그 (2026-04-22)

## 🚀 오늘의 주요 성과
- **OS 통합 기능 구현 완료**: 시스템 트레이, 네이티브 알림, 전역 단축키를 앱 전체에 통합 및 UI와 연결하였습니다.

    - 테마 및 언어 설정을 암호화하여 로컬 디스크에 저장하고 앱 시작 시 자동 복원하는 기능을 구현했습니다.
- **Windows 알림 호환성 확보**: `AppUserModelId` 설정을 통해 Windows 환경에서도 알림이 정상 작동하도록 조치하였습니다.

## 🛠 상세 변경 내역

### 1. 보안 인프라 고도화
- `PersistenceService`를 리팩토링하여 AES-256-GCM 모드 적용 및 Auth Tag 연동.
- 암호화 키 관리 로직 개선 (32바이트 Buffer 강제 조정).

### 2. UI 설정 영구 저장
- `UISettingsService`가 `electronAPI.persistence`를 통해 설정을 비동기로 로드하도록 개선.
- 앱 시작(`bootstrap`) 시 사용자의 테마 및 언어 설정을 암호화 저장소에서 복원.

### 3. OS Core 및 IPC 통합
- `src/main.ts`에 `persistence-set`, `persistence-get` IPC 핸들러 추가.
- 창 닫기('X') 시 트레이로 숨기기 및 단축키 연동 안정화 (Destroyed 에러 방지).
